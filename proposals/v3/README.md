# Dolphin Ergonomics — Proposal v3

## Core Idea
Separate human intent from filesystem structure.

## 1. Human Naming Layer
Users can type long expressive names freely.

- system handles safe storage
- no UI friction
- rename stays instant

## 2. Persistent Selection Memory
Selections persist across navigation.

- held file sets
- reusable state
- inspectable groups

## 3. Spatial Canvas Mode
Files behave like objects in space.

- free arrangement
- clustering by meaning
- no file movement required

## 4. Layout Checkpoints
Save workspace states.

- restore anytime
- multiple views
- non-destructive

## 5. Flow-Based Actions
Replace navigation with direct actions:

- send here
- drop into cluster
- append selection

## 6. Identity Over Path
Files defined by meaning, not location.

- context
- similarity
- visual identity

## Outcome
A spatial, reversible, memory-driven file system layer.
