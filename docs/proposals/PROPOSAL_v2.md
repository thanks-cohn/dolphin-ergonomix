# Dolphin Ergonomics — Proposal v2

## Core Idea
Make Dolphin a fast spatial workspace instead of a folder browser.

## 1. Spatial File Arrangement
Files arranged visually like a collage.

- no file movement
- only layout changes
- grouping by meaning

## 2. Layout Checkpoints
Save and restore arrangements.

- snapshot workspace state
- multiple layouts
- reversible organization

## 3. Timeline Views
View past workspace states.

- history of layouts
- previous sessions
- project states

## 4. Persistent Selections (n0drag)
Selections become reusable state.

- pick up files
- carry across folders
- reuse later

## 5. Clipboard as File Sets
Clipboard becomes reusable collections.

- multiple files
- persistent groups

## 6. Momentum Actions
Replace navigation with actions:

- send here
- move up
- append set

## Outcome
Fast, spatial, reversible file management.
